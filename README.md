
# MeuSite - React + Styled Components

Projeto desenvolvido para estudo de **React Router**, **Styled-Components** e publicação com **GitHub Pages**.

👉 Acesse online:
**[https://Leandro-Pinheiro-Dev.github.io/My-APP-React](https://Leandro-Pinheiro-Dev.github.io/My-APP-React)**

---

## 📸 Sobre o projeto

Este é um site fictício moderno criado com React para praticar:

* Estrutura de páginas (Home, Sobre, Contato)
* Componentização (Header e Footer)
* Estilização com Styled-Components
* Navegação com React Router
* Deploy no GitHub Pages

O objetivo é treinar organização de projetos React do jeito que o mercado utiliza.

---

## 🛠️ Tecnologias utilizadas

* React 18
* React Router DOM v6
* Styled-Components
* Create React App
* GitHub Pages

## ▶️ Como rodar o projeto localmente

Clone o repositório:

```
git clone https://github.com/Leandro-Pinheiro-Dev/my-app.git
```

Entre na pasta:

```
cd my-app
```

Instale as dependências:

```
npm install
```

Inicie o projeto:

```
npm start
```

Abra no navegador:

```
http://localhost:3000
```

## 📦 Build de produção

Para gerar a versão final:

```
npm run build
```

Os arquivos serão gerados na pasta **build/**.

---

## 🌍 Deploy no GitHub Pages

Para publicar o projeto:

```
npm run deploy
```

O site será disponibilizado automaticamente na branch **gh-pages**.
